# サバオッチ 🐱

キジトラ猫「鯖男」を育てるデジタルペットアプリです。

## 🎮 遊び方

| ボタン | 効果 |
|--------|------|
| 🍣 刺身 | おなかを満たす |
| 🪵 爪とぎ | 機嫌をよくする（体力消費） |
| 🧹 そうじ | うんちを片付ける |
| 🌙 ねる | 体力を回復する |

## 🐾 成長ステージ

たまご → こねこ → こども → **鯖男**

放置するとゲージが減っていきます。ゲージがゼロになると…💀
